# bash
`echo $SHELL` to print the shell

## source file
- `. ~/bashrc`

## history
- `history` list
- `!!` execute last command
- `!n` execute command n

## clear
- `ctrl l`

## shortcuts

### line
- `ctrl a` move beginning

### copy / cut / paste
- `ctrl k` cut from cursor to end of line
- `ctrl y` paste last cut text

## infos

### screen
- `tput cols` screen columns
- `tput lines` screen lines

## scripts
- `#!/bin/bash` header

### functions

- `$#` arguments number
- `$@` whole arguments
- `$N` arguments N=1, 2, 3...
- `$0` script
